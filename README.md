# cribl-prometheus-grafana-demo
A Cribl demo sending Prometheus Remote Write metrics into Grafana. 

#Instructions
1. Open Terminal
2. Clone the repo: git clone https://github.com/jlawton-cribl/cribl-prometheus-grafana-demo.git
3. Navigate to the repo folder: cd cribl-prometheus-grafana-demo 
4. Spin up the demo environment: chmod +x ./cribl-prometheus-grafana-demo/start.sh && ./cribl-prometheus-grafana-demo/start.sh
