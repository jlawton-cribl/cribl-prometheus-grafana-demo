# cribl-prometheus-grafana-demo
A Cribl demo sending Prometheus Remote Write metrics into Grafana. 

#Instructions
1. [Install Docker]([url](https://docs.docker.com/engine/install/)) (if you have not already)
2. Make sure Docker is running
   sudo docker run hello-world
3. Create a Prometheus Container
   sudo docker run -p 9090:9090 prom/prometheus
5. You can now access the Prometheus server at http://localhost:9090
